# IPV6

IPv6的地址长度为128位,采用十六进制表示。

 IPv4使用32位(4字节)地址,因此地址空间中只有4,294,967,296(2)个地址。

## 地址写法

### 1、0000可简写为0

### 2、前导0可省略 如0019可写为19

### 3、多段0可写为  ::

2001:0DB8:0000:0023:0008:0800:200C:417A

如

2001:0DB8:0000:0000:0000:0000:200C:417A

等价于

2001:0DB8:0000:0000:0000:0000:200C:417A

等价于

2001:0DB8:0:0:0:0:200C:417A
